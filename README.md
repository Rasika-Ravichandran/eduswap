# 🎓 EduSwap

EduSwap is a student marketplace web application that enables students to buy and sell books, electronics, IoT components, uniforms, and other academic essentials within their campus community.

🚧 Deployment coming soon...

## ✨ Features

- 📚 Browse products by category
- 🔍 Search for products
- 📦 Sell used academic items
- ❤️ Wishlist management
- 💬 Buyer & seller messaging interface
- 👤 Student profile dashboard
- 📱 Responsive design

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Rasika-Ravichandran/eduswap.git
```

### Navigate to the project

```bash
cd eduswap
```

### Install dependencies

```bash
npm install
```

### Run the application

```bash
node app.js
```

Open your browser and visit:

```
http://localhost:3000
```
